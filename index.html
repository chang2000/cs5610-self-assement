Different font

The following 6 lines of text uses font size: 20px, 40px, 50px, 1rem, 2rem and 4rem.

px is a absolute length units and it means pixel. Using px gives you more control on how to display content, and it prevents the problem of text clipped when using relative size font.

rem stands for the font size of the root element. It provides the ability to scale text well for differnt type of diplay devices.
